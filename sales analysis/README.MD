📊 Sales Data Analysis with Pandas & Visualization

This project demonstrates basic data analysis and visualization using Python, Pandas, Matplotlib, and Seaborn.

It works on a small sample dataset of product sales across different cities and months.

🚀 Features
Calculate total sales
Analyze sales by:
City
Product
Month
Visualize data using:
Bar charts
Line plots
Seaborn barplots
🧾 Dataset

The dataset is manually created and includes:

Column	Description
order_id	Unique order ID
city	City of purchase
product	Product name
sales	Sales amount (₹)
month	Month of sale
📦 Technologies Used
Python 🐍
Pandas
Matplotlib
Seaborn
